### Commit Tags

A helper for tagging commits

## Install

```console
$ npm install @cookiescrumbs/commit-tags
```

## Usage

```bash
$ ct
```

```bash
$ commit-tags
```

```
   We tag our commits in the following way:

        type(scope): comment

    **build**: Changes that affect the build system or external dependencies (example scopes:
``` 
